# 🎙️ Speechify 语音选择 - 快速开始

## ⚡ 3 步开始使用

### 1️⃣ 启动 Tataru Assistant

```bash
cd /Users/ruirui/Code/Ai_Code/tataru
npm start
```

### 2️⃣ 选择你的语音

1. 点击设置图标
2. 切换到 **API** 选项卡
3. 找到 **Speechify TTS 设置**
4. 点击 **语音选择** 下拉框：

### 3️⃣ 启用语音引擎

1. 切换到 **窗口** 选项卡
2. 语音引擎：选择 **Speechify**
3. 勾选 **启用语音播放**
4. 点击 **保存**

## ❓ 常见问题

### Q: 如何切换语音？
A: 打开设置 → API → Speechify TTS → 语音选择 → 选择 → 保存

### Q: 需要重启应用吗？
A: 不需要！保存后立即生效。

### Q: 名人语音需要额外付费吗？
A: 不需要！只要你有 Speechify 会员，所有语音都可用。

### Q: 可以为不同角色设置不同语音吗？
A: 目前不支持，但这是一个很好的未来功能建议！
